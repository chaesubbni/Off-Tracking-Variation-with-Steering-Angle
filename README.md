### 프로젝트 개요
- 이 프로젝트는 차량의 조향각 변화에 따라 off-tracking이 어떻게 달라지는지 계산하고 시각화한 실험입니다.
- off-tracking은 앞바퀴와 뒷바퀴 간 경로 차이입니다.
- wmr, 차량의 wheelbase(앞뒤 바퀴 간 거리)와 turning radius(회전 반경)를 이용하여, 조향 시 후륜이 전륜 궤적에서 얼마나 벗어나는지를 그래프로 표현했습니다.
